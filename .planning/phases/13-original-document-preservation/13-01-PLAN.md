Create a new module `src/drive/originals.ts` with the following exports:

1. **`CLIENT_SUBFOLDERS` constant** — array of folder names to pre-create in every client folder:
   ```typescript
   export const CLIENT_SUBFOLDERS = [
     'Income',
     'Property',
     'Down Payment',
     'ID',
     'Originals',
     'Needs Review',
     'Signed Docs',
   ] as const;
   ```
   Derived from Cat's doc categories (DOC_CHECKLIST_RULES_V2 sections) + Originals/ + Needs Review/ per CONTEXT.md.

2. **`preCreateSubfolders(drive: DriveClient, clientFolderId: string): Promise<Record<string, string>>`**
   - Iterates over CLIENT_SUBFOLDERS
   - For each folder name, calls `findOrCreateFolder(drive, name, clientFolderId)` from `../classification/filer.js`
   - Returns a Record mapping folder name -> folder ID (e.g., `{ 'Income': 'abc123', 'Originals': 'def456', ... }`)
   - Uses sequential calls (not parallel) to avoid Drive API rate limits (low volume, simplicity > speed)
   - Log each creation: `[originals] Pre-created subfolder "${name}" (${folderId})`
   - **Does NOT throw** — if an individual folder creation fails, log the error and continue with remaining folders. Return whatever succeeded.

3. **`storeOriginal(drive: DriveClient, clientFolderId: string, pdfBuffer: Buffer, originalFilename: string): Promise<string | null>`**
   - Resolves the Originals/ folder within the client folder: `findOrCreateFolder(drive, 'Originals', clientFolderId)`
   - Generates timestamp-prefixed filename: `${YYYY-MM-DD}_${originalFilename}` where YYYY-MM-DD is today's date (use `new Date().toISOString().slice(0, 10)`)
   - Calls `uploadFile(drive, pdfBuffer, timestampedFilename, originalsFolderId)` from `../classification/filer.js`
   - Returns the uploaded file's Drive ID on success
   - Returns null on any error (log the error, do not throw)
   - This function is write-once: never checks for existing files, never reads back

Import from `../classification/filer.js`: `findOrCreateFolder`, `uploadFile`
Import from `../classification/drive-client.js`: `DriveClient` type
