GHL Tasks API patterns (GoHighLevel V2):
- GET /contacts/{contactId}/tasks — lists all tasks for a contact
- POST /contacts/{contactId}/tasks — creates a task
- PUT /contacts/{contactId}/tasks/{taskId} — updates a task (set completed: true)
- Response shape: { tasks: Array<{ id, title, body, completed, assignedTo, ... }> }
