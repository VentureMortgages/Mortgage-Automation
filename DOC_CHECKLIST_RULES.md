# Document Checklist Rules — Canadian Mortgages

**Status:** DRAFT — Needs Taylor verification
**Purpose:** Define which documents are required based on Finmo application fields
**Last Updated:** 2026-01-22
**Research Sources:** CMHC, WOWA.ca, Ratehub.ca, nesto.ca, Loans Canada, various Canadian mortgage brokers

---

## How This Works

When a Finmo application is submitted, we read these fields and generate a personalized checklist:

| Field | Options | Impact on Checklist |
|-------|---------|---------------------|
| Application Type | Purchase, Refinance | Different property docs required |
| Employment Type | Employed, Self-Employed, Retired, Other | Different income verification |
| Income Sources | Base, Commission, Bonus, Rental, Pension | Additional income docs if applicable |
| Down Payment Source | Savings, RRSP, TFSA, Gift, Sale of Property, Inheritance | Source-specific verification |
| Number of Borrowers | 1 or 2 | Duplicate docs for co-borrower |
| Property Type | Single Family, Condo, Multi-unit, Investment | Property-specific docs |
| Residency Status | Canadian Citizen, PR, Work Permit, Non-Resident | Additional docs for newcomers |
| First-Time Buyer | Yes/No | HBP/FHSA eligibility |
| Prior Bankruptcy/Consumer Proposal | Yes/No | Discharge docs required |
| Separated/Divorced | Yes/No | Legal agreements required |

## SECTION 1: ALWAYS REQUIRED (Every Application)

These documents are required for ALL mortgage applications regardless of type:

| Document | Notes |
|----------|-------|
| Government-issued photo ID | Driver's license OR passport (front + back). Health cards NOT accepted. |
| Second form of ID | Passport, credit card, PR card, SIN card, birth certificate, firearms license |
| Void cheque OR direct deposit form | For mortgage payment setup |
| Signed authorization/consent | For credit check, employment verification, document collection |

## SECTION 2: INCOME DOCUMENTS BY EMPLOYMENT TYPE

### 2A. Employed (Full-Time, Part-Time, Contract)

| Document | Required | Notes |
|----------|----------|-------|
| Recent paystub (within 30 days) | **REQUIRED** | Must show YTD earnings |
| Letter of Employment (LOE) | **REQUIRED** | Must include: position, start date, salary, full-time/part-time, guaranteed hours |
| T4 — Current year | **REQUIRED** | If available (may not be available early in year) |
| T4 — Previous year | **REQUIRED** | For 2-year income history |
| Notice of Assessment (NOA) — Current | If requested | Some lenders require to verify no taxes owing |
| Notice of Assessment (NOA) — Previous | If requested | For income verification |

**Contract/Seasonal Employees — Additional:**
| Document | Required | Notes |
|----------|----------|-------|
| Employment contract | **REQUIRED** | Showing term, rate, renewal likelihood |
| 2 years of T4s | **REQUIRED** | To establish income history |

---

### 2B. Self-Employed (Sole Proprietor, Partnership, Incorporated)

**Standard Self-Employed Documentation (A Lender / Insured):**

| Document | Required | Notes |
|----------|----------|-------|
| T1 General (full tax return) — Current year | **REQUIRED** | Complete return with all schedules |
| T1 General (full tax return) — Previous year | **REQUIRED** | 2-year history required |
| Notice of Assessment (NOA) — Current year | **REQUIRED** | Confirms income reported to CRA |
| Notice of Assessment (NOA) — Previous year | **REQUIRED** | 2-year history |
| Business license/registration | **REQUIRED** | Proof business is active |
| T4 (if paying self a salary) | If applicable | For incorporated individuals |

**Sole Proprietor — Additional:**
| Document | Required | Notes |
|----------|----------|-------|
| T2125 (Statement of Business Activities) | **REQUIRED** | Shows business income/expenses |

**Incorporated — Additional:**
| Document | Required | Notes |
|----------|----------|-------|
| Articles of Incorporation | **REQUIRED** | Proof of corporate structure |
| T2 Corporate Tax Return (2 years) | **REQUIRED** | Corporate financials |
| Accountant-prepared financial statements | **REQUIRED** | Balance sheet, income statement |
| Business bank statements (6-12 months) | If requested | Shows cash flow |

**Stated Income / Bank Statement Program (B Lender / Alt-A):**
| Document | Required | Notes |
|----------|----------|-------|
| Business bank statements (6-12 months) | **REQUIRED** | Shows consistent deposits |
| Personal bank statements (3 months) | **REQUIRED** | Shows personal cash flow |
| Business registration | **REQUIRED** | Proof of active business |
| Signed income declaration | **REQUIRED** | Borrower declares income (must be reasonable for industry) |

*Note: Stated income programs typically require 20-35% down payment and credit score 680+*

---

### 2C. Retired / Pension Income

| Document | Required | Notes |
|----------|----------|-------|
| Pension statement (government or private) | **REQUIRED** | Most recent statement |
| CPP/OAS statement | If applicable | If receiving Canada Pension |
| Notice of Assessment (NOA) | **REQUIRED** | Confirms income |
| RRIF/Annuity statement | If applicable | If receiving payments |
| Investment/dividend income statements | If applicable | If using as qualifying income |

---

### 2D. Other Employment Situations

**Maternity/Parental Leave:**
| Document | Required | Notes |
|----------|----------|-------|
| LOE confirming return date | **REQUIRED** | Must show guaranteed return to work |
| Pre-leave paystub | **REQUIRED** | Shows regular income |
| EI statement | If applicable | Shows current maternity benefits |

**Probationary Period:**
| Document | Required | Notes |
|----------|----------|-------|
| LOE with probation details | **REQUIRED** | Some lenders decline if in probation |
| Previous employment history | **REQUIRED** | Shows stability |

## SECTION 3: VARIABLE/ADDITIONAL INCOME TYPES

### 3A. Commission Income

| Document | Required When |
|----------|---------------|
| T4 history (2 years) showing commission | Commission is part of compensation |
| Letter from employer confirming commission structure | Commission > 20% of total income |
| Commission statements (YTD + previous year) | To verify current earnings |

*Note: Lenders typically average 2 years of commission income. New to commission = harder to qualify.*

---

### 3B. Bonus Income

| Document | Required When |
|----------|---------------|
| T4 history (2 years) showing bonus | Bonus claimed as qualifying income |
| Letter confirming bonus structure | If discretionary vs guaranteed |
| Bonus payment history | To show consistency |

*Note: Discretionary bonuses may not be fully counted. Guaranteed bonuses weighted higher.*

---

### 3C. Rental Income

| Document | Required When |
|----------|---------------|
| T776 (Statement of Real Estate Rentals) | Rental income on tax return |
| Current lease agreement(s) | All rental properties |
| T1 General showing rental income | 2-year history preferred |
| Rental property mortgage statement | If mortgaged |
| Property tax bills for rental | To calculate net rental income |

*Note: Lenders typically use 50-80% of gross rent (offset factor for vacancies/expenses)*

---

### 3D. Child Support / Spousal Support (Receiving)

| Document | Required When |
|----------|---------------|
| Separation agreement OR court order | Using support as income |
| Proof of receipt (bank statements, 12 months) | Must show consistent payments received |
| Divorce decree (if applicable) | Final divorce documentation |

*Note: Must show stable receipt history. Lenders verify support will continue.*

---

### 3E. Other Income Sources

| Income Type | Documents Required |
|-------------|-------------------|
| Disability income | Award letter, recent payment statement |
| Social assistance | Current benefit statement |
| Trust income | Trust documents, payment history |
| Investment/dividend income | Investment statements, T5 slips, 2-year history |

---

## SECTION 4: LIABILITIES & OBLIGATIONS

### 4A. Existing Debts (Always Disclose)

| Document | Required When |
|----------|---------------|
| Current mortgage statement(s) | If owning other properties |
| Line of credit statements | All LOCs |
| Credit card statements | If balances > $0 |
| Car loan/lease statements | Active auto financing |
| Student loan statements | Outstanding student debt |
| Personal loan statements | Any personal loans |

---

### 4B. Support Obligations (Paying)

| Document | Required When |
|----------|---------------|
| Separation agreement OR court order | Paying child/spousal support |
| Proof of payments made (12 months) | Bank statements showing transfers |

*Note: Support payments are deducted from income for qualification.*

---

### 4C. Divorce/Separation

| Document | Required When |
|----------|---------------|
| Separation agreement (signed) | Separated or divorced |
| Divorce decree/certificate | If divorce finalized |
| Property settlement documentation | If dividing assets |
| Spousal buyout agreement | If buying out ex-spouse |

---

### 4D. Prior Bankruptcy or Consumer Proposal

| Document | Required When |
|----------|---------------|
| Certificate of discharge (bankruptcy) | Prior bankruptcy |
| Certificate of full performance (consumer proposal) | Completed consumer proposal |
| Equifax AND TransUnion credit reports | To verify clean since discharge |
| Explanation letter | What caused it, how improved |

**Waiting Periods:**
- CMHC insured: 2 years post-discharge required
- 20%+ down payment: May qualify sooner with B lender
- Must have 2 new credit tradelines, 2 years old, $2,000+ limit each ("2-2-2 Rule")

---

## SECTION 5: DOWN PAYMENT DOCUMENTS

### 5A. Savings / Chequing Account

| Document | Required |
|----------|----------|
| 90-day bank statement history | **REQUIRED** |
| Large deposit explanations | If any deposits > regular income pattern |

*Anti-money laundering: Must show funds were in account 90+ days OR have paper trail for source*

---

### 5B. RRSP (Home Buyers' Plan)

| Document | Required |
|----------|----------|
| RRSP statement (90 days) | **REQUIRED** |
| Form T1036 (HBP Withdrawal Request) | **REQUIRED** |
| Proof of first-time buyer status | See CMHC definition |

**HBP Rules:**
- Max withdrawal: $60,000 per person ($120,000 for couple)
- RRSP funds must be on deposit 90+ days before withdrawal
- Must repay over 15 years (starting year 5 after withdrawal)
- First-time buyer: Haven't owned home in past 4 years

---

### 5C. TFSA

| Document | Required |
|----------|----------|
| TFSA statement (90 days) | **REQUIRED** |

---

### 5D. First Home Savings Account (FHSA)

| Document | Required |
|----------|----------|
| FHSA statement | **REQUIRED** |
| Withdrawal confirmation | **REQUIRED** |

**FHSA Rules:**
- Max contribution: $8,000/year, $40,000 lifetime
- Can combine with HBP on same purchase
- Must be first-time buyer

---

### 5E. Gift Funds

| Document | Required |
|----------|----------|
| Gift letter (signed by donor) | **REQUIRED** — Must state gift is non-repayable |
| Donor's proof of funds | **REQUIRED** — Bank statement showing they have funds |
| Wire transfer confirmation OR bank draft | **REQUIRED** — Proof gift was transferred |
| Relationship declaration | **REQUIRED** — Must be from immediate family |

**Gift Rules:**
- Must be from immediate family (parent, grandparent, sibling)
- Some lenders require donor's ID
- Letter must explicitly state: "This is a gift, not a loan, with no expectation of repayment"

---

### 5F. Sale of Existing Property

| Document | Required |
|----------|----------|
| Accepted offer / sale agreement | **REQUIRED** |
| MLS listing | If applicable |
| Lawyer's statement of adjustments (on closing) | **REQUIRED** — Shows net proceeds |

---

### 5G. Inheritance

| Document | Required |
|----------|----------|
| Estate documents / will | **REQUIRED** |
| Executor letter | **REQUIRED** |
| Bank statement showing receipt | **REQUIRED** |

---

### 5H. Non-Traditional Down Payment (High-Ratio Only)

| Document | Required |
|----------|----------|
| Unsecured LOC or personal loan statement | If using borrowed funds |
| Strong credit history evidence | Required for approval |

*Note: Only available for 90.01%-95% LTV with excellent credit. Must be arm's length, not tied to purchase.*

---

## SECTION 6: PROPERTY DOCUMENTS

### 6A. Purchase Transaction

| Document | Required |
|----------|----------|
| Accepted Offer to Purchase / Agreement of Purchase and Sale | **REQUIRED** |
| MLS listing | If available |
| Feature sheet | If available |
| Home inspection report | Recommended (not always required by lender) |

---

### 6B. Refinance Transaction

| Document | Required |
|----------|----------|
| Current mortgage statement | **REQUIRED** — Shows balance, rate, maturity, lender |
| Property tax bill (most recent) | **REQUIRED** |
| Home insurance declaration page | **REQUIRED** |
| Payout statement (from current lender) | At closing |

---

### 6C. Condo / Strata Properties

| Document | Required | Notes |
|----------|----------|-------|
| Status Certificate (Ontario) / Strata Form B (BC) | **REQUIRED** | Must be dated within 30 days of closing |
| Condo corporation financial statements | Included in status cert |
| Reserve fund study | Included in status cert |
| Declaration, by-laws, rules | Included in status cert |
| Insurance certificate | Included in status cert |

**Status Certificate Red Flags:**
- Ongoing litigation against corporation
- Low reserve fund (< 10% of annual budget)
- Special assessments pending
- High arrears from other owners
- Age of building + major repairs needed

---

### 6D. Multi-Unit Properties (Duplex, Triplex, Fourplex)

| Document | Required |
|----------|----------|
| All current lease agreements | **REQUIRED** |
| Rent roll (list of units, tenants, rents) | **REQUIRED** |
| T776 if currently renting | **REQUIRED** |
| Appraisal (lender ordered) | **REQUIRED** |

**Down Payment Rules:**
- Owner-occupied duplex: 5% minimum (if < $500K)
- Non-owner-occupied: 20% minimum

---

### 6E. Investment / Rental Property (Not Owner-Occupied)

| Document | Required |
|----------|----------|
| All documents from 6D above | **REQUIRED** |
| Proof of other income (not just rental) | **REQUIRED** — Must qualify without full rental income |
| Appraisal | **REQUIRED** |

**Investment Property Rules:**
- 20% minimum down payment (no CMHC insurance available)
- Rental income offset: Lenders use 50-80% of gross rent
- Higher interest rates than owner-occupied

---

## SECTION 7: SPECIAL BORROWER CATEGORIES

### 7A. Newcomers to Canada (Permanent Resident < 5 years)

| Document | Required |
|----------|----------|
| Permanent Resident Card | **REQUIRED** |
| Landing papers (IMM form) | **REQUIRED** |
| Passport | **REQUIRED** |
| Proof of entry date to Canada | **REQUIRED** |
| Canadian employment (3+ months) | **REQUIRED** |
| International credit report OR 12 months Canadian payment history | **REQUIRED** |
| Down payment source verification (may need foreign bank statements) | **REQUIRED** |

**Newcomer Rules:**
- PR status within last 5 years qualifies for newcomer programs
- Some programs waive Canadian credit history requirement
- May need larger down payment (10-20%) depending on credit history
- Foreign income may not be usable

---

### 7B. Non-Permanent Residents (Work Permit)

| Document | Required |
|----------|----------|
| Valid work permit (12+ months remaining) | **REQUIRED** |
| SIN card (starting with 9) | **REQUIRED** |
| Passport | **REQUIRED** |
| Canadian employment letter | **REQUIRED** |
| International credit report or reference letter from home country bank | **REQUIRED** |

**Work Permit Rules:**
- Work permit must have 12+ months remaining
- May require 10-35% down payment depending on lender
- Subject to Foreign Buyer Ban exceptions (check current rules)

---

### 7C. Non-Residents (Foreign Buyer)

| Document | Required |
|----------|----------|
| Passport | **REQUIRED** |
| Proof of foreign income (pay stubs, tax returns, employer letter) | **REQUIRED** |
| International credit report | **REQUIRED** |
| Letter from foreign bank (good standing) | **REQUIRED** |
| Down payment proof (foreign bank statements) | **REQUIRED** |
| Canadian lawyer for ILA | **REQUIRED** |

**Foreign Buyer Rules:**
- **BANNED until 2027** (Prohibition on Purchase of Residential Property by Non-Canadians Act)
- Exceptions: Work permit holders, students, refugees, certain property types
- If eligible: 35%+ down payment typical
- Non-Resident Speculation Tax: 25% in Ontario

---

### 7D. First-Time Home Buyer

| Document | Required |
|----------|----------|
| Declaration of first-time buyer status | **REQUIRED** |
| No property ownership in past 4 years (self or spouse) | Must confirm |

**Benefits:**
- Home Buyers' Plan (RRSP withdrawal up to $60K)
- First Home Savings Account (FHSA)
- 30-year amortization allowed (as of Dec 2024)
- Home Buyers' Tax Credit ($10,000 claim = $1,500 rebate)
- Land Transfer Tax rebates (varies by province)

## SECTION 8: LENDER TYPE REQUIREMENTS

### 8A. A Lender (Banks, Credit Unions — Best Rates)

| Requirement | Threshold |
|-------------|-----------|
| Credit score | 680+ (ideally 720+) |
| Debt ratios | GDS ≤ 39%, TDS ≤ 44% |
| Stress test | Qualify at higher of 5.25% or contract rate + 2% |
| Income verification | Full documentation required |
| Down payment | 5%+ (CMHC insured) or 20%+ (conventional) |

---

### 8B. B Lender (Alternative Lenders — Higher Rates, More Flexible)

| Requirement | Threshold |
|-------------|-----------|
| Credit score | 550-680 (varies) |
| Debt ratios | More flexible (up to 50% TDS) |
| Stress test | May be more lenient |
| Income verification | Bank statements, stated income may be accepted |
| Down payment | Typically 20%+ |
| Term | Usually 1-3 years (then refinance to A lender) |

**When B Lender is needed:**
- Credit score below 680
- Self-employed with hard-to-prove income
- Recent bankruptcy/consumer proposal (< 2 years)
- Non-traditional income sources
- High debt ratios

---

### 8C. Private Lender (Highest Rates, Most Flexible)

| Requirement | Threshold |
|-------------|-----------|
| Credit score | Not primary factor |
| Focus | Property equity (LTV), exit strategy |
| Income verification | Minimal |
| Down payment | 25-35%+ typical |
| Term | 6 months - 2 years |
| Legal | Independent Legal Advice (ILA) required |

**When Private is needed:**
- Very poor credit
- Recently discharged bankruptcy
- No income verification possible
- Bridge financing
- Quick closing required

## SECTION 9: CONDITIONAL LOGIC FOR AUTOMATION

### Rule Engine Pseudocode

```
// ALWAYS REQUIRED
checklist.add("Government photo ID")
checklist.add("Second form of ID")
checklist.add("Void cheque or direct deposit form")
checklist.add("Signed authorization")

// EMPLOYMENT TYPE
IF employment_type == "Employed":
    checklist.add("Recent paystub (within 30 days)")
    checklist.add("Letter of Employment")
    checklist.add("T4 - current year (if available)")
    checklist.add("T4 - previous year")

    IF employment_subtype == "Contract" OR "Seasonal":
        checklist.add("Employment contract")
        checklist.add("2 years of T4 history")

ELSE IF employment_type == "Self-Employed":
    checklist.add("T1 General - current year (full return)")
    checklist.add("T1 General - previous year (full return)")
    checklist.add("Notice of Assessment - current year")
    checklist.add("Notice of Assessment - previous year")
    checklist.add("Business license/registration")

    IF business_structure == "Sole Proprietor":
        checklist.add("T2125 Statement of Business Activities")
    ELSE IF business_structure == "Incorporated":
        checklist.add("Articles of Incorporation")
        checklist.add("T2 Corporate Tax Return (2 years)")
        checklist.add("Accountant-prepared financial statements")

ELSE IF employment_type == "Retired":
    checklist.add("Pension statement")
    checklist.add("Notice of Assessment")
    IF receiving_cpp_oas:
        checklist.add("CPP/OAS statement")
    IF has_rrif_annuity:
        checklist.add("RRIF/Annuity statement")

// VARIABLE INCOME FLAGS
IF has_commission_income:
    checklist.add("T4 history (2 years) showing commission")
    checklist.add("Letter confirming commission structure")

IF has_bonus_income:
    checklist.add("T4 history (2 years) showing bonus")
    checklist.add("Bonus structure confirmation")

IF has_rental_income:
    checklist.add("T776 Statement of Real Estate Rentals")
    checklist.add("Current lease agreements")
    checklist.add("Rental property mortgage statement (if applicable)")

// DOWN PAYMENT SOURCE
IF down_payment_source INCLUDES "Savings":
    checklist.add("90-day bank statements (savings/chequing)")

IF down_payment_source INCLUDES "RRSP":
    checklist.add("RRSP statement (90 days)")
    checklist.add("Form T1036 HBP Withdrawal Request")
    checklist.add("First-time buyer confirmation")

IF down_payment_source INCLUDES "TFSA":
    checklist.add("TFSA statement (90 days)")

IF down_payment_source INCLUDES "FHSA":
    checklist.add("FHSA statement")
    checklist.add("First-time buyer confirmation")

IF down_payment_source INCLUDES "Gift":
    checklist.add("Gift letter (signed, stating non-repayable)")
    checklist.add("Donor's proof of funds")
    checklist.add("Wire transfer or bank draft confirmation")

IF down_payment_source INCLUDES "Sale of Property":
    checklist.add("Sale agreement for existing property")
    checklist.add("Lawyer's statement of adjustments")

IF down_payment_source INCLUDES "Inheritance":
    checklist.add("Estate documents / will")
    checklist.add("Executor letter")
    checklist.add("Bank statement showing receipt")

// APPLICATION TYPE
IF application_type == "Purchase":
    checklist.add("Accepted Offer to Purchase / Agreement of Purchase and Sale")
    checklist.add("MLS listing (if available)")

ELSE IF application_type == "Refinance":
    checklist.add("Current mortgage statement")
    checklist.add("Property tax bill (most recent)")
    checklist.add("Home insurance declaration page")

// PROPERTY TYPE
IF property_type == "Condo":
    checklist.add("Status Certificate (dated within 30 days of closing)")

IF property_type == "Multi-unit" OR "Investment":
    checklist.add("All current lease agreements")
    checklist.add("Rent roll")

// SPECIAL SITUATIONS
IF is_newcomer (PR < 5 years):
    checklist.add("Permanent Resident Card")
    checklist.add("Landing papers")
    checklist.add("Passport")
    checklist.add("International credit report OR 12-month Canadian payment history")

IF has_work_permit:
    checklist.add("Valid work permit (must have 12+ months remaining)")
    checklist.add("SIN card")
    checklist.add("International credit report")

IF is_separated OR is_divorced:
    checklist.add("Separation agreement (signed)")
    IF is_divorced:
        checklist.add("Divorce decree/certificate")
    IF paying_support:
        checklist.add("Proof of support payments (12 months)")
    IF receiving_support:
        checklist.add("Proof of support receipts (12 months)")

IF has_prior_bankruptcy:
    checklist.add("Certificate of discharge")
    checklist.add("Credit reports (Equifax + TransUnion)")
    checklist.add("Explanation letter")

IF has_prior_consumer_proposal:
    checklist.add("Certificate of full performance")
    checklist.add("Credit reports (Equifax + TransUnion)")
    checklist.add("Explanation letter")

// CO-BORROWER
IF num_borrowers == 2:
    // Duplicate all income/employment docs for Borrower 2
    // based on Borrower 2's employment type
    checklist.add("Borrower 2: Government photo ID")
    checklist.add("Borrower 2: [employment docs based on their type]")
```

## SECTION 11: OPEN QUESTIONS FOR TAYLOR

1. **Lender variations:** Do you work primarily with A lenders, or do you also do B lender / private deals? (Affects documentation depth)

2. **Stated income programs:** Do you offer stated income / bank statement programs for self-employed? If so, which lenders?

3. **Commission threshold:** At what % of income do you require the commission confirmation letter? (I assumed 20%)

4. **Gift fund verification:** Do you always require donor's proof of funds, or only for amounts over a certain threshold?

5. **Newcomer programs:** Which banks' newcomer programs do you use most? (TD, RBC, CIBC, Scotiabank have specific programs)

6. **Status certificate:** Do you handle ordering status certificates, or does the buyer's lawyer?

7. **Refinance reasons:** Do you need clients to state their reason for refinancing (debt consolidation, renovation, investment)?

8. **Email turnaround:** What's the expected timeline mentioned in the email? ("Please provide within X business days")

9. **Document portal:** Is there a preferred upload method besides email? (Finmo portal, Google Drive, etc.)

10. **Pre-approval vs full approval:** Does the checklist differ for pre-approval vs full approval?
